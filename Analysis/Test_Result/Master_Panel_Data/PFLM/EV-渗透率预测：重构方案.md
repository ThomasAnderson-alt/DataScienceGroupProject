# 🇬🇧 英国 2035 EV 渗透率预测系统 (Stage 4)

## 终极设计与实施标准指南 (SOP)

### 第一部分：核心模型选择与理论辩护 (Model Selection Justification)
在预测长达 10 年的宏观技术演进时，我们面临多种算法选择。本模型最终选择了**“受限的面板 Logit 回归 (Panel Fractional Logit Model)”**，以下是对其他备选模型的彻底批判与淘汰理由：

1. **为什么不用纯树模型（如 XGBoost / Random Forest）？**
  - **缺陷（时间外推陷阱）：** 树模型本质上是基于历史数据的高维空间切割（If-Else 规则）。它们**绝对不具备时间外推（Extrapolation）能力**。如果强行预测 2035 年，XGBoost 只会将未来的年份全部分配到历史最大年份（如 `year >= 2023`）的叶子节点上，导致未来的“时代红利”变成一条死寂的水平线。
2. **为什么不用传统线性回归（OLS）？**
  - **缺陷（物理穿模）：** 线性回归假设变量间的驱动力是恒定的。这会导致渗透率在 2030 年后轻易突破 100%，甚至飙升至 200%，产生荒谬的物理穿模现象。
3. **为什么不用纯时间序列模型（如 ARIMA / 递归 LSTM）？**
  - **缺陷（指数爆炸与宏观脱钩）：** 时序模型通常基于自身的过去来预测增量（$\Delta Y$）。在长达 10 年的递归推演中，极小的残差噪音会被指数级放大。此外，它们无法吸收“财富 (GDHI)”和“基建 (Ports)”等外部经济变量的结构性突变。
4. **为什么不用巴斯扩散模型 (Bass Diffusion Model)？**
  - **缺陷（同质性假设）：** 巴斯模型（创新者与模仿者模型）虽然能画出完美的 S 曲线，但它通常只能用于“国家级”的宏观同质化聚合预测。它无法在底部融入 LAD 级别的地缘差异（Region底色）、贫富分化和局部基建瓶颈。
5. **🏆 最终选择：带有地缘固定效应的面板 Logit 回归**
  - 结合了机器学习发现的**非线性对数关系**（吸收早期爆炸红利），引入了**面板固定效应 αi​**（吸收不可观测的城市禀赋），并通过 **Logit 空间映射**（死死锁定物理天花板）。

### 第二部分：核心常数推导与严谨文献支撑
本模型通过严格的数学演算与官方文献，钉死了三大物理极值常数。

#### 常数 1：充电基建物理承载极限 ($K_i$)

- **设定公式：**$K_i = \text{veh\_total}_i / 40$ (即每 40 辆汽车配备 1 个公共桩为建设饱和点)
- **文献推导：**
  1. **纯刚需盘 (25%)：** 英国竞争与市场管理局 (CMA) 2021 年最终报告指出，英国有超四分之一（25%）的家庭无路外停车位，绝对依赖公共充电网络。
  2. **车桩服务比 (1:10)：** 依据欧盟 AFID 指令 (2014/94/EU) 与 EAFO 基准，公共车桩成熟服务极限为 10 辆车/桩。
  3. **合成：**$25\% \times (1/10) = 1/40$。该极限在国家宏观层面反算的 82.5 万个桩，完美包容了英国政府预估的 70 万个最终刚需缺口。

#### 常数 2：汽车总盘预测的“人口锚定锁” (Drift Cap)

- **设定公式：** 人车比年化漂移率严格限制在 $\pm0.5\%$ 区间（`np.clip(-0.005, 0.005)`）。
- **理论依据：** 千人汽车保有量是极其稳定的宏观存量指标。限制其漂移，意味着**用人口的重力死死拽住了汽车的数量**，防止回归算法因为老工业区人口流失却算出“人均 3 辆车”的荒谬脱钩灾难。

#### 常数 3：终极渗透率绝对天花板 (`FLEET_CEILING` = 50.24%)

- **常见误区：** 混淆“新车销售渗透率 (Sales)”与“保有量渗透率 (Fleet)”。虽然政策要求 2035 年新车 100% 纯电，但汽车报废周期长达 14 年，马路上依然会充满大量二手燃油车。
- **文献与微积分推导：**
  1. **法律限制：** 英国《2023年车辆排放交易机制令》(ZEV Mandate) 锁定了 2024-2035 每一年的法定销售比例。
  2. **存量盘：** 英国年均新车总销量 (SMMT数据) 约 200 万辆，总汽车存量 3300 万辆，2023 年已有电车存量约 100 万辆。
  3. **积分演算：** 200万 $\times$ 12年ZEV比例之和 (7.79) = 1558万辆新增电车。
  4. **终极天花板：**$(1558\text{万} + 100\text{万}) / 3300\text{万} = \mathbf{50.24\%}$。

### 第三部分：分步实施架构与代码流转

#### 模块一：输入特征动力学沙盘推演 (2026-2035)

1. **人口与财富 (自然低速复利)：** 采用对数线性回归，分别提取并外推 2023 年前与 2019 年前黄金期的低速稳定复利。
2. **车辆总盘 (人车比锚定)：** 提取 2021-2023 的基准人车比，结合 $\pm0.5\%$ 漂移锁，乘以未来人口得出总盘。
3. **基建狂飙 (Logistic 饱和衰减)：** 使用 scipy 的 `curve_fit` 对历史基建进行 Logistic 生长拟合，将 $K_i$ 作为物理饱和极限传入，使得基建在历史冲量结束后平滑着陆。
4. **时空对齐与记忆保留：** 使用遮罩 (Mask) 精准重算 Lag (滞后变量)，同时豁免 2019 年以保留 2018 年的历史真实记忆。

#### 模块二：Logit 空间的主引擎回归与反变换

1. **降维打击与目标映射：**$Y_{i,t} = \ln \left( \frac{P_{i,t}}{50.24\% - P_{i,t}} \right)$
将受限于 50.24% 的现实百分比，映射到 $(-\infty, +\infty)$ 的 Logit 动能空间。
2. **拟合时空引擎：**$Y_{i,t} = \alpha_i + \beta_1 \ln(t - 2010) + \beta_2 \ln(\text{ports}) + \beta_3 \ln(\text{GDHI}) + \beta_4 \ln(\text{pop})$
  - **αi​ (固定效应)：** 吸收 Stage 3 发现的 Region 地缘政策与城市固有限制。
  - **ln(t−2010) (时间衰减)：** 防止年份的线性外推带来永远持续的爆发力幻觉。
  - **Logit 微积分魔法：** 当渗透率逼近 50.24% 时，偏导数 $\partial P/\partial \text{GDHI}$ 会指数级衰减，在数学底层自动筑起了“穷人买不起、普及遇阻”的阶层壁垒，无需人为调节权重。
3. **物理升维与 95% 置信区间 (PI) 提取：**
使用 `expit` (Sigmoid) 将预测均值与误差边界拉回现实，重新获得 $P_{i,t}$。

#### 模块三 & 四：宏观除权聚合与顶级商业可视化

1. **自下而上的无损聚合 (Bottom-Up Roll-up)：**
拒绝直接对 LAD 渗透率求平均值（避免辛普森悖论）。严格遵循：$\sum(\text{纯电车绝对数}) / \sum(\text{汽车总盘})$，向外输出 Country 和 Region 级的宏观数据。
2. **可视化雷达网：**
  - **Country 级带误差带 S 曲线：** 历史(实线)与未来(虚线)分段，加入喇叭状 95% PI 误差阴影，十字刻度瞄准 2030/2035，剥离数据标注以符合顶刊 Data-Ink 规范。
  - **Region 级 12 轨并发图：** 极简细线布局，精准呈现“大区起飞时间差”。
